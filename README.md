Code for The Statistics Bootcamp course of my master's degree.

# How To Run

Prerequisites:
* R version ```4.0.4``` or later
* renv version ```0.12.5``` or later

### Development

1. Run ```make init``` to initialize the environment.
2. Run ```make run``` to execute the script.
